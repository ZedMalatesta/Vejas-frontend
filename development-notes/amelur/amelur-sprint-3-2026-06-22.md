# Sprint 3: Directives, Pipes & Forms (@angular/core, @angular/forms)

**What was done:**
What was done:
В рамках данного спринта была начата разработка страницы профиля пользователя. На данный момент подготовлена базовая
структура страницы и прорабатывается отображение информации о пользователе.
Также продолжилось знакомство с Signals и RxJS. Постепенно становится понятнее, в каких случаях лучше использовать
каждый из подходов для работы с состоянием и асинхронными данными.
Помимо практической части уделялось время подготовке к теории спринта: изучались материалы по управлению состоянием,
Change Detection и реактивному программированию в Angular.

**Problems:**
Пока остаются вопросы по организации состояния приложения и выбору подходящего инструмента для разных сценариев.
Иногда сложно определить, где будет достаточно Signals, а где логичнее использовать RxJS.
Также при проектировании страницы профиля возникли вопросы по структуре данных и дальнейшему расширению
функциональности.
Не понятно до сих пор будет ли реализован полноценный backend для приложения или можно обойтись supabase.

**Solutions:**
По мере разработки страницы профиля планируется постепенно внедрять изученные подходы и оценивать их на реальном коде
проекта.
